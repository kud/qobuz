**@kud/qobuz** is a typed, ESM-only client for Qobuz's (reverse-engineered) API.
It wraps search, rich metadata, favourites and playlists, and deep links into the
Qobuz apps — returning clean camelCase domain types with zero runtime
dependencies. It's the shared core that [`@kud/qobuz-cli`](/projects/qobuz-cli)
builds on.

## 🔑 How authentication works

Qobuz has no public API and guards login behind a captcha, so `@kud/qobuz` never
handles passwords. Instead you give it a `user_auth_token` borrowed from a
logged-in browser session, and a pluggable credential store keeps it — in-memory,
the macOS Keychain, or your own implementation. Every request rides on that
token; the library never sees a password.

## 📦 Install

```sh
npm install @kud/qobuz
```
